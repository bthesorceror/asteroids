My toy game project! FUN STUFF HERE!

## Important notes

### Install tools

```
sudo npm install -g gulp nodemon
```

### Install dependencies

```
npm install
```

### Starting asset compilation for development

```
gulp watch
```

### Manually building assets

```
gulp build
```

## Starting server

Currently server will be running on port 8080

### Development

```
nodemon
```

### Otherwise

```
npm start
```
